# NSS Book 4 - Controllers, Authentication, Authorization
## Project 1: :bike: Bianca's Bike Shop
Bianca's Bike Shop has asked us to build an application that Bianca, her mechanics, and bike owners can use to manage bicycle repairs.
Topics introduced: Controllers, Authentication, Authorization, Role-based Access Control
